# 🚀 Daily Utility Planner

A minimalist yet powerful productivity hub built with vanilla JavaScript. This tool combines a **Pomodoro Timer**, a **Smart Todo List**, and a **Custom Music Player** to create the perfect focused work environment.

---

## ✨ Key Features

* **🍅 Pomodoro Timer:** 25-minute focus intervals with Start/Pause/Reset functionality and automatic mode switching (Work/Break).
* **✅ Todo List:** Manage your tasks with ease. Features include adding, completing (strike-through), and deleting tasks.
* **🎵 Smart Music Player:** * **Pre-loaded Focus Track:** Start focusing instantly with a built-in ambient track.
    * **Custom Upload:** Upload your own `.mp3` files locally to stay in the flow.
    * **Volume Control:** Precision audio slider to set the perfect background atmosphere.
* **💡 Motivational Tips:** A curated collection of rotating tips across Coding, Chess, Music, and Life.
* **💾 Local Storage:** Your tasks are automatically saved to your browser, so you don't lose them when you refresh.
* **🌍 Clean UI:** Fully English interface with a modern, responsive design and intuitive controls.

---

## 🛠 Tech Stack

* **HTML5** - Semantic structure.
* **CSS3** - Custom styling with a focus on usability and modern look.
* **JavaScript (ES6+)** - Core logic for timer, music player, and storage.
* **Vercel** - Hosting and seamless deployment.
